-Este ficheiro serve de orientação para o que cada ficheiro representa.

é necessário ter os modulos cryptography, timeit na sua versao mais recente há data de 20/3/2024


-"b_c_d.py" tem a implementação dos pontos B) C) e D) respetivamente, mas não inclui a alteração para automatizar os resultados.


-"dados_testes.txt" contém os resultados dos testes com mais pormenor


-"source_code.py" tem todo o código emplementado e está pronto a correr, com uma pequena interface que permite testar os resultados, este encontra-se automatizado para imprimir na linha de comando os resultados.
	
	A interface é simples, o ficheiro vai criar automaticamente os ficheiros necessarios para correr.
	Depois tem uma opção de escola: 1 para testar o método AES (B)
					2 para testar o método RSA (C)
					3 para testar o método SHA (D)
					-1 para fechar o programa
			Também é possível em cada escolha decidir entre usar sempre os mesmos ficheiros (escolha 1)
								 	testar com diferentes ficheiros (escolha 2)
